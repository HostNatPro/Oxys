# Oxys

## Docker quick install

Install Docker Engine on Linux with one command:

```bash
curl -fsSL https://raw.githubusercontent.com/HostNatPro/Oxys/main/scripts/docker.sh | bash
```

The script detects your OS, installs Docker from the official repositories, and shows progress with a step-by-step spinner. Installation logs are hidden unless something fails.

You can also run it locally:

```bash
./scripts/docker.sh
```
